# 🎯 GD Arena – AI-Powered Group Discussion Practice Platform

GD Arena is an AI-powered full-stack web application that helps users improve their communication and speaking skills through real-time group discussions with AI participants. The platform provides instant AI responses, tracks user performance, and maintains daily streaks to encourage consistent practice.

## 🌐 Live Demo

🚀 **Live Website:** https://gd-arena-cgh4.vercel.app/

---

## ✨ Features

- 🤖 Practice group discussions with multiple AI participants.
- 💬 Real-time AI-powered conversations using Groq AI.
- 🔐 Secure user authentication with Firebase Authentication.
- 📊 Performance dashboard with discussion analytics.
- 🔥 Daily streak tracking and progress monitoring.
- 📜 Discussion history for reviewing previous sessions.
- 📱 Responsive and user-friendly interface.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/abhay963/GD-ARENA-frontend.git
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file and add the required Firebase, Groq AI, and database credentials.

Example:

```env
VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
GROQ_API_KEY=your_api_key
DATABASE_URL=your_database_url
```

### Run the Project

```bash
npm run dev
```
